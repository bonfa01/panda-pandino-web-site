# Panda a Pandino 

Sito web ufficiale del raduno **Panda a Pandino**, il più grande evento al mondo dedicato alla **Fiat Panda**.

Il progetto ha l'obiettivo di presentare l'evento, raccontarne la missione e permettere agli utenti di rimanere aggiornati sulle novità, iscrizioni e attività del raduno.

---

# 🌍 Descrizione

Il sito include diverse sezioni informative tra cui:

- Hero section con presentazione dell'evento
- Missione del raduno
- Etica e solidarietà
- Archivio fotografico delle edizioni passate
- Pagina News
- Pagina iscrizioni
- Sezione contatti e social

---

# 🛠 Tecnologie utilizzate

Il progetto è sviluppato utilizzando:

- **HTML5**
- **CSS3**
- **JavaScript**
- **PHP**
- **Git**

---

# 📂 Struttura del progetto

- `index.html`: Homepage del sito.
- `news.html`: Sezione dedicata alle ultime novità dal raduno.
- `iscrizioni.html`: Modulo e informazioni per i Pandisti.
- `css/`: Contiene tutti i fogli di stile .
- `source/`: contiene le immagini e loghi.

---

## 🚀 Installazione

### 1. Clonare il repository

```bash
git clone https://github.com/tuo-username/panda-a-pandino.git
````

### 2. Entrare nella cartella del progetto

```bash
cd panda-a-pandino
```


### 3. Aprire il browser

Aprire il browser e andare su:

```
http://localhost:8000
```
